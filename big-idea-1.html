Big Idea 1: Creative Development

Explore how computing innovations are developed through collaboration and diverse perspectives, including program design, development, and documentation.

Topic 1.1
Collaboration
Explore how collaboration affects the development of computing innovations and how computing innovations can reflect diverse perspectives.

Key Skills:

  • Explaining how collaboration affects the development of a program
  • Identifying diverse perspectives that influence the development of a program
  • Understanding collaboration strategies and tools
Topic 1.2
Program Function and Purpose
Learn to identify the purpose of a computing innovation and explain how a program's statements correspond to its purpose.

Key Skills:

  • Determining the purpose of a computing innovation
  • Explaining how a program functions
  • Identifying inputs, outputs, and program components
Topic 1.3
Program Design and Development
Understand program design and development processes, including how programs are developed and maintained through iterative cycles.

Key Skills:

  • Developing programs using an iterative process
  • Creating program documentation
  • Acknowledging code segments from other sources
Topic 1.4
Identifying and Correcting Errors
Learn how to identify and correct errors in algorithms and programs, including testing methodologies and debugging techniques.

Key Skills:

  • Identifying errors in algorithms and programs
  • Correcting syntax and logic errors
  • Testing programs with appropriate test cases
